# ICPDB Goal Audit

This document maps the product goal to current evidence.

## Product Goal

Build an Internet Computer SQLite DBaaS:

- storage: `ic-sqlite-vfs`
- UI feel: Supabase Table Editor
- CLI/API feel: Turso shell and inspect
