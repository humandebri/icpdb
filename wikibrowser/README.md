# ICPDB Console

Next.js console for the ICPDB canister.

## Local

```bash
pnpm install
cp .env.local.example .env.local
pnpm dev
```

Open:

```text
http://localhost:3000/icpdb
```

Required env:

```bash
NEXT_PUBLIC_WIKI_IC_HOST=http://127.0.0.1:8001
NEXT_PUBLIC_II_PROVIDER_URL=http://id.ai.localhost:8001
NEXT_PUBLIC_ICPDB_CANISTER_ID=<local-icpdb-canister-id>
```

Wallet approve defaults:

```bash
NEXT_PUBLIC_ICPDB_WALLET_SIGNER_URL=https://oisy.com/sign
NEXT_PUBLIC_ICPDB_WALLET_HOST=https://icp-api.io
```

## Scope

- Internet Identity login
- create/list databases
- SQL query/update execution
- usage and billing display
- ICP deposit quote, wallet approve, deposit execution
- payment history
- owner token creation

## Checks

```bash
pnpm test
pnpm typecheck
pnpm lint
pnpm build
```

`next-env.d.ts` is generated by Next and is intentionally ignored. `pnpm typecheck` runs `next typegen` before `tsc`.

## Cloudflare Workers

The frontend can be deployed through OpenNext:

```bash
pnpm build:worker
pnpm deploy
```

Set build variables for browser-side values:

```bash
NEXT_PUBLIC_WIKI_IC_HOST=https://icp-api.io
NEXT_PUBLIC_II_PROVIDER_URL=https://identity.ic0.app
NEXT_PUBLIC_ICPDB_CANISTER_ID=<mainnet-icpdb-canister-id>
NEXT_PUBLIC_ICPDB_WALLET_SIGNER_URL=https://oisy.com/sign
NEXT_PUBLIC_ICPDB_WALLET_HOST=https://icp-api.io
```
